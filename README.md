# Components App

A simple React component-based webpage featuring a header, banner, image gallery, and footer.

## Live App

View the published app here:

https://olabayode.github.io/components/

## Features

- Styled header with navigation links
- Simple banner section with call-to-action buttons
- Gallery section with four equally sized images
- Footer with static content, contact details, and newsletter subscribe button
- Responsive layout for smaller screens

## Technologies Used

- React
- JavaScript
- CSS
- Create React App
- GitHub Pages
